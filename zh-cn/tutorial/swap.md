# Swap
（以$MASK举例）
点击右上角的connect，选择你的钱包。
 

连接钱包后，返回主页面，选择“资产”。
* $MASK 与 $USDC 交易对。
* 请确保您有足够的 $ETH 来支付钱包中的 gas。
* 如果您选择其他资产，Balancer 将自动转换为 $USDC，然后交易为 $MASK。额外的步骤可能会导致交易费用的增加。

![](../asset/26.jpeg)

输入 $MASK 的合约地址：0x69af81e73a73b40adf4f3d4223cd9b1ece623074
请确保您输入正确的地址。
